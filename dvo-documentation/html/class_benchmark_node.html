◆ run()

void BenchmarkNode::run ( )

The most important funciton in BenchmarkNode.

  1. setup the visualizer
  2. generate id for each frame in the VideoFolder.
  3. (configure debugging visualizer)
  4. setup camera parameters (create intrinsicMatrix)
  5. setup configuration for tracker node (class CameraDenseTrackerConfig) from server, and output in ROS warning.
  6. initialize first pose
  7. read all rgb-d pair entries
  8. start iteration: 8.1 create Reference Camera and do something about visualization 8.2 if Estimate is Required, then call dense_tracker.match(*reference, *current, relative)
  9. if ShowEstimate or RenderVideo is set in cfg_, then do it.
