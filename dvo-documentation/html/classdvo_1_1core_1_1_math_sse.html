idea:

vec = [j0 j1 j2 j3 j4 j5]

vec * vec.transpose() = [ j0*j0 j0*j1 j0*j2 j0*j3 j0*j4 j0*j5 j1*j0 j1*j1 j1*j2 j1*j3 j1*j4 j1*j5 j2*j0 j2*j1 j2*j2 j2*j3 j2*j4 j2*j5 j3*j0 j3*j1 j3*j2 j3*j3 j3*j4 j3*j5 j4*j0 j4*j1 j4*j2 j4*j3 j4*j4 j4*j5 j5*j0 j5*j1 j5*j2 j5*j3 j5*j4 j5*j5 ]

row multiplicators:

j0 * [j0 j1 j2 j3 j4 j5] j1 * [j0 j1 j2 j3 j4 j5] j2 * [j0 j1 j2 j3 j4 j5] j3 * [j0 j1 j2 j3 j4 j5] j4 * [j0 j1 j2 j3 j4 j5] j5 * [j0 j1 j2 j3 j4 j5]

substitute:

v1 = [j0 j1 j2 j3], v2 = [j4 j5 j0 j1], v3 = [j2 j3 j4 j5]

calculate:

[j0 j0 j0 j0] * v1, [j0 j0 j1 j1] * v2, [j1 j1 j1 j1] * v3 [j2 j2 j2 j2] * v1, [j2 j2 j3 j3] * v2, [j3 j3 j3 j3] * v3 [j4 j4 j4 j4] * v1, [j4 j4 j5 j5] * v2, [j5 j5 j5 j5] * v3

the vectors for multiplication can be obtained by shuffling v1 and v2 (they are prescaled with scale to v1s and v2s)

after each multiplication add result to matrix
